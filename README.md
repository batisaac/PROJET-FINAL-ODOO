# IT Parc — Module Odoo 18 BATOUA ISAAC

## Installation
1. Copier le dossier `it_parc` dans le répertoire `addons/` d'Odoo
2. Redémarrer le serveur Odoo
3. Activer le mode développeur (Paramètres → Mode développeur)
4. Mettre à jour la liste des applications
5. Rechercher "IT Parc" et cliquer sur Installer
